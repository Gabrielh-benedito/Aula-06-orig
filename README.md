# Sistema de Gestão de Biblioteca (POO)

Projeto desenvolvido em Python utilizando:

- Encapsulamento
- Herança
- Polimorfismo
- Organização em múltiplos arquivos

## Como executar

1. Baixe o projeto
2. Abra no VSCode
3. Execute:

python main.py

## Funcionalidades

- Cadastro de Livro
- Cadastro de Revista
- Listagem de Itens
- Empréstimo de Item
- Devolução de Item

## Estrutura

- item_biblioteca.py → Classe base
- livro.py → Subclasse Livro
- revista.py → Subclasse Revista
- biblioteca.py → Classe de controle
- main.py → Menu principal
